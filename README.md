## 「だだんだんと空飛ぶバイキンUFO」
このCGアニメーションでは、『それいけ！アンマンパン』に登場する「だだんだん」と、バイキンマンが操る縦空飛ぶUFOを3DCGで再現した。 青空と草原が広がる穏やかな情景に、視覚として登場するだだんだんは、手を動的に動かし、アンパンマン立ち向かう。その頭上には、バイキンマンのUFOが奇妙な軌道を描きながら飛び回る。<br/>
